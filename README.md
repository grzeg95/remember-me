# Remember Me
firebase based project

## Dev init
Before developing make sure to run:
- `npm install`
- `npm install ./functions`
- `gcloud auth login`
- `gcloud config set project remember-me-3`

##Dev use
- `npm run start`
- `npm run serve:functions`

##Deploy hosting
- `npm run deploy:hosting`

##Deploy functions
- `npm run deploy:functions`
